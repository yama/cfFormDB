## 設置

### インストール

1. `/assets/modules/` フォルダ内に `cfFormDB` フォルダをアップロードします。
2. MODX管理画面にログインし、モジュール管理画面を開きます。
3. 「モジュールを作成」ボタンをクリックし、以下の情報を入力します。（「説明」は任意）
   - モジュール名： `cfFormDB`
   - 説明： **1.0.1** cfFormMailerで送信された情報を表示・CSV出力
   - モジュールコード欄： ファイル `module.cfFormDB.php` の内容をコピー＆ペースト
4. 「設定」タブをクリックし、「モジュール設定」欄に以下の文をコピー＆ペーストし、「パラメータ表示の更新」ボタンをクリックします。
   ```
   &viewFields=一覧画面で表示する項目;text;
   ```
5. 「一覧画面で表示する項目」欄に、一覧表示画面で表示するフィールド名をコンマ区切りで入力します。（※任意）
   フィールド名はcfFormMailerで使用しているフィールド名となります。
   指定しない場合は、すべての項目が表示されます。
6. 「保存」ボタンをクリックし、モジュールの登録を完了します。
7. モジュールタブから「cfFormDB」をクリックします。
8. 初回起動時は「cfFormDB用のテーブルが作成されていません。」と表示されますので、画面内の「テーブル新規作成」ボタンをクリックします。
   cfFormDB用のテーブルがデータベースに追加されます。
9. 「テーブルを作成しました」と表示され、設置完了です。

### 利用開始

cfFormMailerの環境設定チャンクから、`use_store_db` の設定値を「1」にします。

```
# 投稿内容をデータベースに記録する （0: 記録しない[デフォルト] / 1: 記録する）
# この機能を使用するには、あらかじめcfFormDBモジュールをインストールし、記録用テーブルを作成しておく必要があります
use_store_db = 1
```

## 機能解説

### 表示

情報が保存されている場合、cfFormDBモジュールタブをクリックすると、登録が新しいものから順に一覧表示されます。
一画面に表示する件数を変更することができます。（初期値：30件）

「全項目表示」をクリックすると、該当する登録の登録内容一覧が表示されます。

### CSV出力

「CSV出力」ボタンをクリックすると、出力する項目と出力件数、出力順を指定する画面が表示されます。
出力したい条件を指定したら、「出力する」ボタンをクリックして下さい。
